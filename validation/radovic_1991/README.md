# Radović (1991) — S/F Superlattice Validation

## Citation
Z. Radović, M. Ledvij, L. Dobrosavljević-Grujić, A.I. Buzdin, and J.R. Clem,
"Transition temperatures of superconductor-ferromagnet superlattices,"
*Physical Review B*, vol. 44, pp. 759–764, 1991.

## Status
**Implemented.** Validates Tc(d_F) curve for Nb/Ni superlattice using
thin-S proximity model (EQ-4). Parameters: Tc0=9.2 K, d_S=20 nm,
ξ_S=38 nm, ξ_F=2.3 nm, E_ex=75 meV, γ=0.3.
