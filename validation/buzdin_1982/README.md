# Buzdin (1982) — S/F Proximity Effect Validation

## Citation
A.I. Buzdin, L.N. Bulaevskii, and S.V. Panyukov,
"Critical-current oscillations as a function of the exchange field and
thickness of the ferromagnetic metal (F) in an S-F-S junction,"
*JETP Letters*, vol. 35, pp. 178–180, 1982.

## Equations Reproduced
- **Pair amplitude**: Eq. (2) — F(x) = F₀ exp(-x/ξ_F) cos(x/ξ_F)
- **Tc suppression**: Single-mode approximation for Tc(d_F) showing oscillatory behavior.

## Figures
- **tc_vs_df.csv**: Tc(d_F) curve for Nb/Fe bilayer showing non-monotonic Tc suppression
  and partial recovery (reentrant behavior).
- **pair_amplitude.csv**: F(x) profile at d_F = 10 nm showing decaying oscillation in the F layer.
