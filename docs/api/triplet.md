# supermag.triplet

Spin-triplet superconductivity solver for magnetic multilayers. Computes
long-range equal-spin triplet correlations generated by non-collinear
magnetization at interfaces.

**C++ engine:** `triplet.cpp`

---

## `triplet.solve()`

Compute spin-triplet pair correlations in a magnetic multilayer.

```python
supermag.triplet.solve(
    n_layers, thicknesses, magnetization_angles,
    n_grid=200, xi_F=1.0, xi_N=10.0
)
```

> **C++ note:** The C++ engine also supports temperature dependence (`T`),
> per-layer exchange energies (`E_ex_per_layer`), per-layer diffusion
> coefficients (`D_per_layer`), and a mode enum
> (`PHENOMENOLOGICAL`/`DIFFUSIVE`). These are not yet exposed in the
> Python API.

### Parameters

| Name | Type | Default | Description |
|------|------|---------|-------------|
| `n_layers` | int | *required* | Number of layers |
| `thicknesses` | array_like | *required* | Layer thicknesses (nm), shape `(n_layers,)` |
| `magnetization_angles` | array_like | *required* | Magnetization direction per layer (rad), shape `(n_layers,)` |
| `n_grid` | int | `200` | Total spatial grid points |
| `xi_F` | float | `1.0` | Short-range (ferromagnetic) coherence length (nm) |
| `xi_N` | float | `10.0` | Long-range (triplet) coherence length (nm) |

### Returns

| Type | Description |
|------|-------------|
| `x` : `numpy.ndarray` | Position array (nm) |
| `f_triplet` : `numpy.ndarray` | Equal-spin triplet pair amplitude $\|f_{\uparrow\uparrow}(x)\|$ |

### Physics

Three pair types exist in S/F structures:

| Component | Range | Decay |
|-----------|-------|-------|
| Singlet $f_0$ | Short | $e^{-x/\xi_F}\cos(x/\xi_F)$ |
| Triplet $m_s = 0$ ($f_3$) | Short | $e^{-x/\xi_F}$ |
| Equal-spin triplet $f_{\uparrow\uparrow}$ | **Long** | $e^{-x/\xi_N}$ |

Equal-spin triplets are generated at interfaces where the magnetization
rotates between adjacent layers (EQ-12):

$$f_{\uparrow\uparrow} \propto \sin(\alpha)\, e^{-|x - x_\text{int}|/\xi_N}$$

where $\alpha$ is the misalignment angle. Maximal at 90° misalignment.

### Example

```python
import numpy as np
import supermag

x, f_trip = supermag.triplet.solve(
    n_layers=3,
    thicknesses=[10.0, 5.0, 10.0],
    magnetization_angles=[0.0, np.pi/2, np.pi],
    n_grid=300, xi_F=1.0, xi_N=10.0
)
# Peaks at interfaces where magnetization angle changes
```
