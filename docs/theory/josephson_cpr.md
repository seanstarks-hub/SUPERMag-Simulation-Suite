# Josephson Current-Phase Relations

Josephson junctions with ferromagnetic barriers (S/F/S) exhibit anomalous
current-phase relations, including 0-π transitions where the ground-state
phase difference switches from 0 to π.

## Key Equations

**Josephson current:**
$$I(\phi) = I_c \sin(\phi + \phi_0)$$

**Critical current in S/F/S (Buzdin, simplified):**
$$I_c(d_F) \propto \exp(-d_F/\xi_F) \cos(d_F/\xi_F - \pi/4)$$

The 0-π transition occurs when $I_c$ changes sign at $d_F \approx 0.7\xi_F$.

- Golubov, A.A. et al., Rev. Mod. Phys. **76**, 411 (2004).
